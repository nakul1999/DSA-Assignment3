Heap 

A heap data structure is a complete binary tree used to efficiently manage elements based on priority.

In a min heap, the parent node is always smaller than its children,
so the smallest element is always at the root.
In a max heap, the parent is larger than its children,
so the largest element is at the root.

Heaps are commonly implemented using arrays and are mainly used in priority queues,
scheduling systems, and algorithms like Dijkstra’s shortest path.

They allow fast insertion and deletion of the highest/lowest priority element in O(log n) time,
while accessing the root element takes O(1) time.
